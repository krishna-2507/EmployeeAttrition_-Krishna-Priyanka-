# Employee Attrition Prediction using Machine Learning

## Project Overview
This project predicts whether an employee is likely to leave the company using machine learning techniques. The analysis is based on the IBM HR Analytics Employee Attrition dataset and focuses on identifying the factors that contribute to employee attrition.

## Dataset
* IBM HR Analytics Employee Attrition Dataset
* Total Records: 1,470 employees

## Project Workflow
* Data Loading and Exploration
* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Model Building
* Model Evaluation
* Feature Importance Analysis
* Business Recommendations

## Machine Learning Models
* Logistic Regression
* Random Forest Classifier
* Gradient Boosting Classifier

## Best Model
Random Forest performed the best based on the evaluation metrics and was selected as the final model.

## Top Factors Influencing Attrition
* Monthly Income
* Age
* Daily Rate
* Total Working Years
* Years at Company
* Years with Current Manager
* Overtime
* Monthly Rate
* Hourly Rate
* Distance from Home
